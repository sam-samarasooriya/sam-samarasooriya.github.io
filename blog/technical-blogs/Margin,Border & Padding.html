Difference between Margin, Border and Padding

  • What is CSS Box Model

    In CSS the The Box Model define how to display space between differet elements in a web page.

    Websites are created a set of boxes and these contains four main components as shown below.


  • What is the difference between margin and padding in CSS?

    The main difference between padding and margin is that padding controls how much breathing
    room exists within a box, while margin controls how much breathing room or whitespace exists outside of a box

    1. Margin
      • Location: Outermost space, outside the border.
      • Impact: Affects the distance between this element and surrounding elements, pushing other elements away.
      • Example: If you have two boxes with margins, the margin will determine how far apart the boxes are from each other.
    2. Border
      • Location: Between the padding and margin.
      • Function: Defines the outline of the element, creating a visible boundary.
      • Impact: Adds a line around the element, which can be styled with different colors, widths, and styles (e.g., solid, dashed).
      • Example: If you apply a border to a box, it will appear as a line around the element, separating it from the margin and padding.
      3. Padding
      • Location: Inner space, between the content and the border.
      • Function: Adds space inside the element, pushing the content away from the edges of the border.
      • Impact: Increases the space between the content of the element and its border, without affecting the space outside the element.
      • Example: If you add padding to a text box, the text will move inward from the edges of the box, creating extra space inside the border.
